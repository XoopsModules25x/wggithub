Hilfe: wgGitHub

Beschreibung

Das Modul wgGitHub kann zum Anzeigen von GitHub-Inhalten der XOOPS-Verzeichnisse verwendet werden

Installieren/Deinstallieren

Es sind keine speziellen Maßnahmen erforderlich, führen Sie den Standardinstallationsprozess aus und extrahieren Sie den Ordner wggithub in das Module-Verzeichnis. Installieren Sie das Modul über Admininstration -> System -> Module.

Wenn Sie zustätzliche Informationen benötigen, lesen Sie die Installationshinweise unter XOOPS Operations Manual

Funktionen

Das ModuleBuilder-Modul wird weiter ausgebaut, um die Bedingungen für die Erstellung von Modulen zu erfüllen, die immer ausgefeilter werden.
Aus diesem Grund lade ich alle Entwickler ein, Änderungen oder Ergänzungen an diesem Modul zu melden und einzusenden, damit wir gemeinsam zur Entwicklung beitragen können
